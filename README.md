### What I learned

In this project I learned how to wrap and center content using flexbox

see below:


```css
.container{
    width: 100vw;
    height: 80vh;
    background-color: var(--Very_light_gray_background);
    display: flex;
    align-items: center;
    justify-content: center;
    flex-wrap: nowrap;

}

```
